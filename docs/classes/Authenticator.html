    Authenticator Class

    Extends Ember-Simple-Auth.Authenticators.BaseAuthenticator
    Defined in: app\authenticators\authenticator.js:6
    Module: security
    Parent Module: explorviz

    This Authenticator sends a single AJAX request with data fields "username" and "password" to the backend. The backend checks the authentication data and responds with a randomized token, if authentication was successful. This token is now used for all future requests by the Authorizer, since all backend resources are secured by token-based authorization. authenticate is called by Login-Form/authenticate:method.
